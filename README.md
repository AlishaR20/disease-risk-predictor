# disease-risk-predictor# 🧬 VitalSense AI — Disease Risk Predictor

An AI-powered web app that predicts health risks based on lifestyle data like diet, activity, and sleep patterns.

## 🔗 Live Demo
👉 https://alishar20.github.io/disease-risk-predictor/

## 🎯 Problem It Solves
Most people don't know their health risks until it's too late.
VitalSense AI analyzes your daily lifestyle habits and predicts
risk across 8 major disease categories — instantly.

## ✨ Features
- 🏥 Predicts risk for 8 diseases (Diabetes, Cardiovascular, Mental Health & more)
- 🤖 Powered by Claude AI for personalized health insights
- 📊 Visual risk score bars with severity levels
- 💡 Prioritized preventive recommendations (P1/P2/P3)
- 🔒 No data stored — 100% private
- 📱 Fully responsive — works on mobile

## 🧠 How It Works
1. User enters lifestyle data (diet, sleep, exercise, stress etc.)
2. AI engine scores 23+ health indicators
3. Claude AI generates personalized insights & recommendations
4. Results shown instantly with visual risk breakdown

## 🏗️ Tech Stack
- HTML, CSS, JavaScript (zero dependencies)
- Claude AI API (Anthropic) for AI insights
- GitHub Pages for deployment

## 🚀 Run Locally
Just download index.html and open it in any browser. No setup needed!
