🎯 Objective
An AI-powered Disease Risk Predictor that analyzes lifestyle data such as diet, physical activity, and sleep patterns to predict possible health risks and provide personalized preventive health recommendations instantly.

---

✅ Expected Outcomes Achieved
✅ AI-powered disease risk prediction
✅ Personalized preventive health recommendations
✅ AI-driven health insights across 8 disease categories
✅ Visual risk score dashboard with severity levels
✅ Privacy-first — no health data stored

---

🛠️ Tech Stack

| Frontend -> HTML, CSS, JavaScript 
| AI Engine -> Claude AI API (Anthropic) 
| Risk Scoring -> Custom ML-style scoring engine 
| Deployment ->GitHub Pages 

---

⚙️ How It Works
1. 👤 User enters lifestyle data — diet, sleep, activity, stress, BMI etc.
2. 🧠 AI engine analyzes 23+ health indicators in real time
3. 📊 Risk scores generated across 8 disease categories
4. 🤖 Claude AI produces personalized insights & recommendations
5. 📋 User receives a complete visual health risk report instantly

---

🌟 Project Highlights
- ⚡ Instant analysis — results in under 3 seconds
- 🏥 Covers 8 diseases — Cardiovascular, Diabetes, Mental Health & more
- 🔒 100% private — all processing done locally, no data stored
- 📱 Fully responsive — works on mobile and desktop
- 💡 Prioritized recommendations — P1, P2, P3 action plan
